# One-paged infographics

## How to link your image?

1. Create an infographic-style image: Ensure the file format is .png or .jpeg.

2. Naming Convention: Your file must be named exactly as follows (e.g., Reading01_infographic.png, Reading04_infographic.png).  
   📌 _Note that filenames are case-sensitive._

4. Upload the file: Add your image to the designated folder in this repository.
5. Verify on Streamlit: Check our Streamlit page to see if your image is displayed correctly.
